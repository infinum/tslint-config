# tslint-config-react

## Installation

NPM:
```bash
npm install @infinumjs/tslint-config-react
```

Yarn:
```bash
yarn add @infinumjs/tslint-config-react
```

## Support

```json
{
  "extends": "@infinumjs/tslint-config-react"
}
```
