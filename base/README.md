# tslint-config

## Installation

NPM:
```bash
npm install @infinumjs/tslint-config
```

Yarn:
```bash
yarn add @infinumjs/tslint-config
```

## Support

```json
{
  "extends": "@infinumjs/tslint-config"
}
```
